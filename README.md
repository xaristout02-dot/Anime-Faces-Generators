#Anime Face Generation: VAE vs DCGAN

A full project that compares two deep generative models — a Variational Autoencoder (VAE) and a Deep Convolutional GAN (DCGAN) — trained from scratch to generate synthetic anime face images, using the Anime Face Dataset (https://www.kaggle.com/datasets/splcher/animefacedataset).


Project for the MSc in Applied Mathematics & Machine Learning, University of Thessaly (2025–2026).
